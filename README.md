## Pre-requisites

### .env File

Create a .env file in the root directory with the following variables:

```sh
DATABASE_USER=postgres
DATABASE_PASSWORD=123456
DATABASE_HOST=localhost
DATABASE_PORT=5432
DATABASE_NAME=postgres
```
